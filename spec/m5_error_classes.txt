# m5 ordinal parser/comparator error classes (j5)
# Anchor: design/meta_engine/2026-04-26_m5_i1i2_parser_comparator_omega_cycle.json axis j5
# Stamp: Γ_0+1 ; closed catalog — adding a new class requires ω-cycle amendment.

ParseError                    : malformed lexeme or unexpected token at position P; structured msg includes (token, pos, expected_set).
EmptyInput                    : input string is empty or whitespace-only after NFC normalisation; subclass of ParseError but distinct dispatch.
NonCanonicalCNFError          : grammar parses but a side-condition (SC1..SC4) fails: non-strict-descending exponents, zero coefficient, +0 summand, or sentinel-as-summand.
OrdinalBeyondPredicativeError : comparator asked to compare two non-sentinel terms involving Γ_0/ψ-arithmetic; collapsing function would be required (j8).
UnknownSentinel               : sentinel-shaped lexeme that is not in the closed set {ε_0, Γ_0, ψ(Ω_ω)}; e.g. ψ(Ω_42), φ_2(0).
NumericOverflow               : nat literal exceeds parser's BigInt budget OR exponent depth > MAX_NESTING (config-bound); raised before CNF construction.
HomoglyphRejected             : input contains Latin/Cyrillic look-alike for ω/ε/Γ/ψ/Ω, or Unicode whitespace U+00A0/U+2009/U+200B (LX2/LX4).
SentinelArithmeticError       : alias of OrdinalBeyondPredicativeError raised on 'sentinel + tail' or 'sentinel · n' input shapes (SN3); kept distinct for catalog clarity.
RoundTripMismatch             : property-test internal — parse(print(α)) ≠ α; signals serializer drift, not a user-facing parse failure.
DepthBudgetExceeded           : recursive descent exceeded configured depth (default 64); guards against pathological 'ω^(ω^(ω^...))' nesting attacks.
